Grammar

Since the Calvinese language acts like an alphabetical code, English grammar remains largely the same when translating. However there are some key differences. When a single letter repeats within a word a single apostrophe replaces the following letter.

For example:

Test --> ti'tsti; instead of tiitsti

When a 'chunk' repeats (like the Calvinese letter E) the repeating chunk is replaced with a "

For example:

Book --> beou"kh; instead of beououkh

Yet, with words that end in the same letter (like see or tree), what apostrophe to use can be ambiguous. However, the " should be used since the chunk is repeating.

For example:

See --> tsi"; instead of tsii


Calvinese punctuation mirrors English punctuation. However, since a singular apostrophe and quotation marks are used as grammar indicators in Calvinese, French guillemets are used when quoting someone or something. Since each letter in the Calvinese alphabet consists of two English letters, abbreviations are translated into Calvinese using the first letter of the corresponding Calvinese letter.

Translator

Calvinese is typically written in all lowercase, so despite the input, the output will be lowecase. The translator will immediately convert the quotes to guillemets. All other English punctuation will be preserved. Most importantly, do not use contractions, translations will be incorrect.
